**Кваліфікаційна робота на підтвердження ступеня фахового молодшого бакалавра.**

---

## 📌 Про проєкт
**EduSync** - це спеціалізована клієнт-серверна вебсистема, створена для автоматизації процесів організації, супроводу, контролю та документообігу виробничої і навчальної практики здобувачів освіти. Система усуває недоліки традиційного паперового обліку та забезпечує швидку і зручну взаємодію між адміністрацією, викладачами та студентами.

## ⚙️ Використані технології
* **Backend:** Python 3, Flask (мікрофреймворк)
* **База даних:** SQLite (з використанням параметризованих запитів для захисту від SQL-ін'єкцій)
* **Frontend:** HTML5, CSS3, Jinja2 (Server-Side Rendering)
* **Безпека:** `werkzeug.security` (хешування паролів pbkdf2:sha256)
* **Тестування:** Pytest

## 🔑 Основний функціонал (Рольова модель)

### 🛡️ Адміністратор (Методист)
* Додавання та редагування інформації про бази практик.
* Керування обліковими записами студентів та викладачів.
* Формування наказів (закріплення студентів за підприємствами та керівниками).
* Перегляд загальної зведеної статистики.

### 👨‍🏫 Викладач (Керівник від коледжу)
* Моніторинг закріплених студентів.
* Перегляд електронних щоденників практики.
* Рецензування фінальних звітів (додавання коментарів, відправка на доопрацювання).
* Виставлення підсумкових оцінок.

### 👨‍🎓 Студент
* Перегляд інформації про місце проходження практики та керівника.
* Ведення електронного щоденника практики (інтерактивний календар).
* Завантаження звітних документів.
* Відстеження статусу перевірки («Призначено», «У процесі», «Надіслано», «Оцінено»).

# Pereddiplomna
Переддипломна робота 2026: Система моніторингу студентських практик
# 🎓 Система обліку проходження практик студентами

Веб-застосунок для автоматизації та моніторингу виробничої практики студентів. Проєкт розроблений як дипломна та переддипломна практична робота для спеціальності **121 "Інженерія програмного забезпечення"**.

## 📌 Про проєкт

Ця система дозволяє координувати взаємодію між адміністрацією коледжу, керівниками практик від навчального закладу та студентами. Вона замінює паперові щоденники та звіти зручним веб-інтерфейсом.

### 🛠 Технологічний стек
* **Backend:** Python + Flask
* **Database:** SQLite3
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Templates:** Jinja2

## 👥 Ролі користувачів

### 👑 Адміністратор
- Керування базою студентів, викладачів та підприємств.
- Призначення студентів на бази практики з вказанням термінів та тем.
- Перегляд загальної статистики та статусів звітів.

### 👨‍🏫 Керівник практики (Викладач)
- Перегляд списку закріплених студентів.
- Моніторинг щоденників практики в реальному часі.
- Перевірка фінальних звітів та виставлення оцінок.

### 🎓 Студент
- Перегляд інформації про місце практики та завдання.
- Щоденне заповнення записів у щоденнику.
- Подача підсумкового звіту для перевірки.
